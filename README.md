
# SuperKart MLOps Sales Forecasting Project

This repository contains the complete MLOps pipeline for the SuperKart sales forecasting project.

## Project Components

- Data registration on Hugging Face Dataset Hub
- Data preparation and train-test split
- Model training and evaluation
- Final model registration on Hugging Face Model Hub
- Flask backend deployment files
- Streamlit frontend deployment files
- GitHub Actions CI/CD workflow
